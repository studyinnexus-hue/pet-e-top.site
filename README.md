# Pet é Top 🐾

Plataforma educativa para conscientização sobre bem-estar animal, combate aos maus-tratos e histórias reais ilustradas.

## 🚀 Sobre o Projeto
O **Pet é Top** é um portal de notícias e histórias focado na educação de jovens e crianças sobre o respeito aos animais. O site utiliza Inteligência Artificial (Google Gemini) para gerar conteúdos dinâmicos e informativos.

## ✨ Funcionalidades
- **Radar Pet:** Notícias reais e atualizadas sobre o mundo animal com links para as fontes originais.
- **Pet's Storybook:** Histórias emocionantes de superação geradas por IA, com narração em áudio e ilustrações exclusivas.
- **Aba Escola:** Guia prático com ideias de projetos (Teatro, Livros, Cartazes) para alunos transformarem suas escolas em ambientes amigos dos animais.
- **Design Editorial:** Interface inspirada em grandes portais como CNN e Google, garantindo clareza e autoridade.

## 🛠️ Tecnologias Utilizadas
- **React 18** + **Vite**
- **Tailwind CSS** (Estilização)
- **Motion** (Animações)
- **Lucide React** (Ícones)
- **Google Gemini API** (IA para textos, áudio e imagens)

## 📦 Como Rodar o Projeto
1. Clone o repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/pet-e-top.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure sua chave da API no arquivo `.env`:
   ```env
   GEMINI_API_KEY=sua_chave_aqui
   ```
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
